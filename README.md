#TouchGame 2D
-Javaで作ったゲームソフトウェアです。
-概要：
-キャラクターを使って障害物を回避してゴールを目指すゲームです。
-使い方：
画面タッチでキャラクターを動かし、右から来る、様々な障害物を避けます。
キャラクターからエネルギーを発射し、障害物を破壊することができます。
破壊した障害物でスコアがアップします。
キャラクターが障害物に触れると、ゲームは終了します!



実行環境:

-Build Tools Version 30.0.3
